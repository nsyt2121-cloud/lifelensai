# Project Name 
LIFELENS AI

## Problem Statement
People often struggle to make important life decisions (career, studies, finance, daily planning) due to lack of personalized guidance, clarity, and structured thinking. Existing tools provide generic advice but do not adapt to individual situations or offer real-time intelligent support.

There is a need for an AI-powered system that can analyze user inputs, understand personal context, and provide smart, actionable decision-making assistance in an interactive and user-friendly way.

## Project Description
LifeLens AI is an intelligent decision-support system designed to help users make better life choices in real time. By combining user inputs with AI-powered analysis, the system provides personalized suggestions, predictions, and action plans for everyday decisions such as career planning, productivity, finance, and mental well-being.
